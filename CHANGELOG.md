### Breaking Changes

- From

```js
const validator = new Validator({
  input,
  rules,
  customMessage: {},
  customAttributes: {},
  locale: 'en',
  confirmedReverse: true,
})
```

- to

```js
const options = {
  customMessage: {},
  customAttributes: {},
  locale: 'en',
  confirmedReverse: true,
}
const validator = new Validator(input, rules, options)
```
